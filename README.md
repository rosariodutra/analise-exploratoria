# 🔍 Análise Exploratória de Dados (EDA)

EDA completa sobre dataset de vendas utilizando Python e SQL (SQLite). Identificação de padrões, tendências, rankings e geração automática de insights — tudo no terminal.

  → Priorizar Licença ERP em campanhas de venda
  → Replicar estratégias do mês de Março

  → Analisar práticas da vendedora Ana para treinamento

## ✨ Funcionalidades

- **Dataset gerado automaticamente** — 200 registros de vendas (2024)
- **Visão geral** — contagem, receita total, ticket médio, min/max
- **Receita por categoria** com gráfico de barras no terminal
- **Ranking de vendedores** com medalhas e ticket médio
- **Tendência mensal** com variação percentual Jan→Dez
- **Receita por região** com distribuição percentual
- **Insights automáticos** — padrões e recomendações gerados via SQL
- **Banco de dados SQLite** com queries SQL reais
